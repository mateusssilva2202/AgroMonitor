# AgroMonitor – Sistema de Monitoramento Agrícola

O **AgroMonitor** é um sistema web voltado para o monitoramento e automação de estufas agrícolas, integrando conceitos de **IoT**, **Segurança da Informação** e **Computação em Nuvem**.

Atualmente o projeto foca na autenticação segura de usuários, preparando a base para integração com sensores físicos utilizando ESP32.

---

##  Funcionalidades

-  Sistema de login seguro  
-  Cadastro de usuários com criptografia  
-  Bloqueio automático após 5 tentativas de login incorretas  
-  Controle de tentativas falhas  
-  Páginas de recuperação de senha  
-  Interface web responsiva  
-  Termos de Política de Segurança integrados  

---

##  Tecnologias Utilizadas

- Python 3  
- Flask  
- PostgreSQL  
- HTML5  
- CSS3  
- JavaScript  
- bcrypt (criptografia de dados)
